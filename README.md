# E-Commerce Store

A simple E-Commerce Store built using Flask and SQLite.

## Features
- User Registration
- User Login
- Product Listing
- Add to Cart
- Shopping Cart
- Checkout
- Place Orders
- Admin Dashboard
- Add Products

## Technologies Used
- Python
- Flask
- SQLite
- HTML
- CSS

## How to Run

1. Install the required packages:
   pip install -r requirements.txt

2. Run the application:
   python app.py

3. Open:
   http://127.0.0.1:5000/
